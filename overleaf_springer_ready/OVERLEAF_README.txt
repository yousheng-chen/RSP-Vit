Overleaf-ready Springer template

1. Upload this folder as a zip to Overleaf.
2. Set `main.tex` as the Main document.
3. `svproc.cls`, `aliascnt.sty`, `remreset.sty`, `svind.ist`, and `svindd.ist` are already in the root folder.
4. Use the `templates` folder only as reference/sample material.

If you want to add figures, keep them in a folder such as `figures/` and reference them from `main.tex`.
